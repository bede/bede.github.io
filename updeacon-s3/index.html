updeacon-s3 alpha

Securely dehost and upload DNA sequences in fasta/fastq format directly to S3 object storage. Runs Deacon using WebAssembly to remove human sequences prior to upload. FASTQ files are uploaded to a timestamped directory prefix in the specified bucket. Nested directory structures are preserved.

Usage
  1. Enter S3-compatible object storage credentials.
  2. Press 'download index' to fetch a human Deacon index. You should only need to do this once. Alternatively, drag & drop a copy of the panhuman-1 index.
  3. Drag & drop a directory containing FASTQ/FASTA sequences you wish to upload.
  4. Press either 'Filter & upload' or 'Filter only'.
