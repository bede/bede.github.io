updeacon-s3 alpha

Dehost and upload fasta/fastq file(s) to CLIMB S3 object storage. Runs Deacon locally using WebAssembly to remove human sequences prior to upload.

Usage
  1. Enter the access key ID and secret access key shown in the 'S3 Buckets' area of the CLIMB portal
  2. Drag and drop a local copy of the panhuman-1 Deacon index (850 MB)
  3. Drag and drop the directory of sequences you wish to upload
  4. Press upload and await completion
